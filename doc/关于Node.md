## Node

简单的说 Node.js 就是运行在服务端的 JavaScript。
Node.js 是一个基于 Chrome JavaScript 运行时建立的一个平台。

### 现在的前端跟 Node 有什么关系

- 我们要使用`npm`来进行包管理（`npm`是随同 NodeJS 一起安装的包管理工具）
- 现在的前端开发环境需要基于 `node`

### 安装

https://nodejs.org/en/

从官网下载 node 的稳定版安装到本地

```
➜  ~ node -v
v8.11.1
➜  ~ npm -v
5.6.0
```

安装完成后尝试在  控制太输入`node -v`、`npm -v`来查看版本，如果能正常显示，那么说明 node 安装成功了
